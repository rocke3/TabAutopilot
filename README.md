# TabAutopilot — AI-Powered Tab Manager for Chrome

> Stop drowning in tabs. TabAutopilot automatically organizes, names, and cleans up your browser tabs using on-device AI — with zero data sent to any server.

---

## Features

- **Smart Grouping** — AI reads your tab titles and groups them by topic, not just domain
- **Duplicate Detection** — badge shows duplicate count; one click closes them all
- **Tab Hibernation** — inactive tabs discarded to free memory; reload on demand
- **Workspaces** — save, restore, and switch between complete tab sessions (groups preserved)
- **Memory Dashboard** — track estimated memory saved with 30-day history
- **Activity Scoring** — visual indicator of how active each tab is
- **Recently Closed** — last 8 closed tabs with one-click restore
- **Undo** — last 10 actions are undoable with `Cmd/Ctrl+Z`
- **Keyboard Shortcuts** — `Alt+G` group, `Alt+D` dedup, `Alt+S` save workspace
- **Context Menu** — right-click any page to group, hibernate, or deduplicate

**Everything runs on-device using Gemini Nano. Your browsing data never leaves your browser.**

## Privacy

- No account required
- No data collection
- No external API calls
- AI runs entirely on-device via Chrome's built-in Gemini Nano
- All data stored locally in your browser
